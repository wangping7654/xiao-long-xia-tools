## 常见问题记录

### 安装问题
**Q**: 如何安装xiao-long-xia-tools?
**A**: `pip install git+https://github.com/wangping7654/xiao-long-xia-tools.git`

**Q**: 依赖安装失败?
**A**: 确保Python版本≥3.8，使用虚拟环境

### 使用问题
**Q**: 如何查找重复文件?
**A**: 使用`FileUtils.find_duplicate_files(directory)`

**Q**: 数据清洗的最佳实践?
**A**: 参考examples/data_cleaning_example.py

### 贡献问题
**Q**: 如何提交PR?
**A**: Fork仓库，创建分支，提交PR，通过CI测试

**Q**: 代码规范是什么?
**A**: 遵循PEP 8，使用black格式化
