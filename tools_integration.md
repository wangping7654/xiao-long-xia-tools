# 🦞 关键工具集成配置

## 已安装工具

### 1. Claude-Mem (已安装)
**版本**: 最新版 (通过npm安装)
**类型**: 持久化记忆系统
**状态**: ✅ 安装完成

#### 核心功能
- 自动捕获编码会话中的所有操作
- AI压缩和总结会话内容
- 相关上下文注入未来会话
- 搜索工具和Web查看器 (http://localhost:37777)

#### 配置步骤
1. **启动Claude-Mem服务**:
   ```bash
   claude-mem start
   ```

2. **配置OpenClaw集成**:
   - 需要设置环境变量或配置文件
   - 启用记忆持久化插件

3. **验证安装**:
   ```bash
   claude-mem status
   ```

### 2. OpenViking (安装中)
**版本**: 0.2.5 (通过pip安装)
**类型**: AI Agent上下文数据库
**状态**: ⏳ 安装进行中

#### 核心功能
- 统一管理记忆、资源、技能
- 文件系统范式组织上下文
- 分层上下文加载 (L0/L1/L2)
- 目录递归检索
- 可视化检索轨迹
- 自动会话管理

1. **创建配置文件** (`~/.openviking/ov.conf`):

       "level": "INFO",

       "dense": {

## 集成架构

### 记忆管理三层架构
```
原始记忆 → Claude-Mem (捕获和压缩) → OpenViking (结构化存储) → 各Agent使用
```

### Agent访问流程
1. **实时响应Agent**接收用户请求
2. **主控协调Agent**分配任务
3. **专业Agent**通过OpenViking检索相关记忆
4. **执行结果**通过Claude-Mem记录
5. **重要记忆**存入OpenViking长期存储

### 成本优化策略
1. **分层加载**: OpenViking的L0/L1/L2分层减少token使用
2. **智能压缩**: Claude-Mem的AI压缩减少存储需求
3. **精准检索**: 目录递归检索提高命中率
4. **渐进披露**: 按需加载详细信息
